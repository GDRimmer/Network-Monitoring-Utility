NMAP Network Scanner

NMAP (Network Mapper) is a powerful open-source tool for network discovery and security auditing. Our interface provides:

  • Multiple Scan Types: Run simple port scans, comprehensive scans with OS detection, focused service detection, and more
  • Visual Results: Easy to read reports with visual representation of discovered hosts and services
  • Scan History: Review past scans and compare network changes over time
  • Custom Arguments: Advanced users can specify custom NMAP command arguments

Packet Capture (TCPDump)

Monitor network traffic at the packet level using TCPDump, a powerful command-line packet analyzer:

  • Interface Selection: Capture traffic on any network interface
  • Filtering Options: Filter by protocol (TCP, UDP, ICMP), port numbers, or specific hosts
  • Downloadable Captures: Save captures in PCAP format for analysis in tools like Wireshark
  • Live Monitoring: View capture progress in real-time

ARP Table

View and manage your system's Address Resolution Protocol (ARP) table, which maps IP addresses to MAC addresses on your local network:

  • Live ARP Table: View all current ARP entries on your system
  • Vendor Identification: Automatically identify hardware vendors from MAC addresses
  • ARP Manipulation: Ping specific hosts to update ARP entries
  • Export Options: Export ARP table data in CSV or JSON formats

Network Security Tips

These tools are designed for network administrators to monitor and diagnose their own networks. Always ensure you have proper authorization before scanning or monitoring any network.
Best Practices:
  • Regularly scan your network to discover unauthorized devices
  • Keep an inventory of expected devices and their MAC addresses
  • Monitor for unusual traffic patterns using packet captures
  • Document your network topology and expected routes
  • Maintain a baseline of normal network performance
  • Use strong passwords and encryption on all network devices
